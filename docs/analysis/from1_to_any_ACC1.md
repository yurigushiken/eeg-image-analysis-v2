- GFP-based collapsed localizer analysis with 72 subject-condition combinations.

- P3b: Peak at 476 ms, FWHM window [366.1, 538.1] ms (width: 171.9 ms)
- Fz: Peak at 88 ms, FWHM window [74.9, 104.5] ms (width: 29.6 ms)
